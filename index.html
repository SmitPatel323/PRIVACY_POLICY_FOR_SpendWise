Privacy Policy for SpendWise

Effective Date: 02 March 2026

Developer: Smit Patel

1. Introduction

SpendWise is a personal expense tracking mobile application developed and maintained by Smit Patel. We respect your privacy and are committed to protecting your personal information.

2. Information We Collect

  • Email address (for account creation and login)
  • User name
  • Financial data such as transactions, categories, amounts, and notes
  • Basic app usage data required for functionality and performance

3. How We Use Your Information

The information collected is used only to:

  • Create and manage user accounts
  • Authenticate users securely
  • Store and display personal expense data
  • Improve app performance and user experience

We do not sell, trade, or rent your personal data to third parties.

4. Data Storage and Security

SpendWise uses Google Firebase services for authentication and secure cloud data storage. Access to data is protected using Firebase Authentication and security rules.

5. Data Sharing

Your data is not shared with third parties except:

  • When required by law
  • When necessary for Firebase services to operate

6. User Control

Users can manage their data directly within the app, including adding or deleting transactions. Users may stop using the app at any time.
